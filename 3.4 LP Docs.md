## üî§Font Family Change

We‚Äôve integrated a new typography stylesheet that manages:

- Font sizes  
- Font weights  
- Overall font styling  

The global font family is now set to **"Calibri, sans-serif"**, and all LPs built with the 3.4 template will automatically inherit this style.

---

## üé®New Section Background Classes

To improve visual consistency, the following **section background classes** have been added:

- `section-bg-primary`  
- `section-bg-secondary`  
- `section-bg-white`  
- `section-bg-grey`  
- `section-bg-darkblue`  

These classes automatically adjust the **text color** based on the background.

## üîòNew Button Styles

We‚Äôve added a variety of **button styles**. Apply any of the following classes to an anchor (`<a>`) tag to style them as buttons:

<img src="./assets/img/buttons.png" alt="Archer Card Assets Preview" style="max-width: 500px; max-height: 400px;border-radius:8px;" />

### Primary Button Classes

- `primary-button`
- `secondary-button`
- `button-darkblue`
- `button-clear`
- `button-clear-black`

### Text Link Styles

- `text-link`
- `text-link-white`
- `text-link-black`

### Icon Add-ons (Optional)

To include icons within buttons, use these additional classes:

- `new-window`
- `right-arrow`
- `video-play`
- `download`
- `chat-later`
- `chat-now`
- `contact-us`
